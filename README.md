# Enterprise Cloud Risk Management Assessment

## Overview
This project applies the NIST Risk Management Framework (RMF) to a simulated government AWS environment supporting HR, financial, and monitoring applications.

## Skills Demonstrated
- Enterprise Risk Assessment
- Security Categorization
- NIST RMF
- NIST SP 800-53
- System Security Plan (SSP)
- POA&M Recommendations
- Continuous Monitoring
- AWS Cloud Governance
- IAM, CloudTrail, CloudWatch, and S3

## Key Deliverables
- Security Categorization
- System Security Plan
- AU-2 Audit Events Control Assessment
- Assessment Plan
- Continuous Monitoring Strategy
- POA&M Summary

## Technologies & Frameworks
- NIST RMF
- NIST SP 800-53
- NIST SP 800-60
- AWS IAM
- AWS CloudTrail
- Amazon CloudWatch
- Amazon S3

## Project Summary
I assessed a simulated government AWS environment by identifying system boundaries, categorizing information types, evaluating risk, documenting selected controls, and developing monitoring strategies to support governance and compliance objectives.

## Note
This is an academic simulation project. No real government systems, sensitive data, or production environments were used.
